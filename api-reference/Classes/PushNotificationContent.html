PushNotificationContent

@objc(NBLPushNotificationContent)
public class PushNotificationContent : NSObject

The uneditable content of a notification. Contains the details of the notification, including the type of notification that was delivered, any custom data you stored in the userInfo dictionary, and any attachments.
